# Project Sylvanas – Universal Dispels (Reference)

Optional sub-plugin of **Universal Utility**. Install from Marketplace, then:

**Main Menu → Universal Utility → Universal Dispels**

Automatically detects and removes harmful debuffs in **PvP** and **PvE** based on configurable logic.

---

## Core settings

| Setting | Description |
|--------|-------------|
| **Priority Filter** | How aggressively dispels are prioritized: **Low** / **Medium** / **High** |
| **Combat Mode** | **PvP only** / **PvE only** / **Both** |
| **Humanizer Preset** | Simulates reaction time: **Slow** / **Medium** / **Fast** |
| **Allow Drawings** | Toggles in-game visual feedback for dispel events |

---

## Built-in spell database

Works out of the box with a curated list of dangerous debuffs per class and encounter type. Users can report missing or misclassified debuffs for refinement.

---

*Reference for HCS rotations: when using Universal Dispels, rotation scripts can avoid duplicating dispel logic (e.g. Cleanse Toxins, etc.) or document that users should configure dispels there.*
